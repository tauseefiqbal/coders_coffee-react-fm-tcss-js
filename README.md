# Coffee App

A modern coffee shop website built with React, Vite, Tailwind CSS and Framer Motion.

## Features
- Responsive design
- Hero section
- Services showcase
- Where to buy section
- App banner
- Footer
- Modular React components

## Tech Stack
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Project Structure

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```

## Customization
- Update images in `src/assets/coffee/` and `src/assets/website/`.
- Edit component files in `src/components/` to change content or layout.
